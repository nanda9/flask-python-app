🚀 CI/CD DevOps Pipeline

GitHub → Docker → ArgoCD → Kubernetes

Git Push triggers the GitHub Actions CI/CD pipeline.

Docker builds the application image and pushes it to Docker Hub.

ArgoCD monitors the Git repository and synchronizes the Kubernetes cluster.

Kubernetes performs a rolling update and deploys the latest application version.
